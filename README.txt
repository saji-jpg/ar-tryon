制服AR試着（Chromebook版）
============================

使い方
1. このフォルダーをHTTPS対応のWebサーバーへアップロードします。
2. ChromebookのChromeで index.html のURLを開きます。
3. 「カメラを開始」を押し、カメラの使用を許可します。
4. 画面下の「夏服」「合服」「冬服」で制服を切り替えます。

注意
- カメラは、セキュリティ上 localhost または HTTPS のページでのみ使用できます。
- 初回起動時はMediaPipeを読み込むためインターネット接続が必要です。
- 映像はブラウザ内だけで処理し、サーバーへ送信しません。
